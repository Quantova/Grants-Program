# Contributing

Thank you for your interest in the Quantova Grants Program. This repository tracks
grant **applications**, **RFPs**, and **milestone deliveries**.

## Submitting a grant application

1. Read the [program process](docs/process.md) and the
   [application template](applications/application-template.md).
2. Fork this repository.
3. Copy `applications/application-template.md` to
   `applications/your_project_name.md` (a **new file** — do not modify the template).
4. Fill it out and open a pull request.
5. A grants committee member will review your PR, may request changes, and will
   discuss next steps with you.

You can also apply at **https://quantova.org/grants/** or email
**Grants@quantova.org**.

## Suggesting an RFP

See [rfps/README.md](rfps/README.md). Copy `rfps/suggestion-template.md` to a new
file and open a pull request.

## Delivering a milestone

If your grant is approved, follow [docs/milestone-delivery.md](docs/milestone-delivery.md).

## Pull request etiquette

- One application per pull request.
- Name your file after your project; don't edit the template files.
- Keep discussion in the pull request so the history stays transparent.
- Be responsive to committee feedback.

## Code of conduct

All participants must follow the [Code of Conduct](CODE_OF_CONDUCT.md).

## Questions

Email **Grants@quantova.org** or visit **https://quantova.org/grants/**.
