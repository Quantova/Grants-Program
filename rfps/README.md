# Requests for Proposals (RFPs)

Requests for Proposals are concrete project ideas that the Quantova ecosystem would
like to see built. There are two ways this folder is used:

1. **Published RFPs** — ideas the grants team has written up and would like a
   capable team to implement. If you want to work on one, apply with a normal
   [grant application](../applications/application-template.md) and indicate the RFP
   on the first line of your project overview.
2. **Suggestions** — anyone can suggest an idea for the ecosystem to build, even
   without the team to build it.

## Suggesting an idea

1. Fork this repository.
2. Copy `suggestion-template.md` to a new file in this folder named after your idea:
   `your_idea.md`. Do not edit the template file directly.
3. Fill it out and open a pull request.

If the team believes the idea is useful, it may be turned into a published RFP and
matched with a team. You can also email **Grants@quantova.org** with suggestions, or
submit via **https://quantova.org/grants/**.
