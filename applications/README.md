# Applications

This folder holds Quantova grant applications. Each accepted and pending application
is a Markdown file named after the project.

## Submitting an application

1. Read the [program process](../docs/process.md) and the
   [application template](application-template.md).
2. Fork this repository.
3. Copy `application-template.md` to a new file in this folder named after your
   project: `your_project_name.md`. **Do not edit the template file itself.**
4. Fill it out and open a pull request.

You can also apply on the website at https://quantova.org/grants/ or email
Grants@quantova.org with questions.
